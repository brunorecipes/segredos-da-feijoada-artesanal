## Tecnologia e restrições

- Saída: **apenas um arquivo HTML completo** (`<!DOCTYPE html> ... </html>`).  
- Usar **apenas Tailwind CSS via CDN**, com:

  ```html
  <script src="https://cdn.tailwindcss.com"></script>
  ```

- Não usar React, Vue, Next.js ou qualquer outro framework JS.  
- Pode JavaScript inline apenas para coisas simples  
  (por exemplo, animação de scroll). Mas priorize sempre Tailwind.
- Estrutura **responsiva (mobile-first)**, bem legível em smartphones.  
- Layout premium e moderno, com bastante respiro, boa hierarquia visual e visual profissional.

As imagens devem ser referenciadas na pasta images, por exemplo:

- Logo: `/images/logo.png`  
- Mockup do produto (PDF/áudios): `/images/mockup-produto.png`  
- Foto do especialista: `/images/especialista.png`
